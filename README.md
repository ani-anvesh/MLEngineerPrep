# Machine Learning Specialization: 14-Day Sprint

## Overview

This repository documents a 14-day hands-on sprint to complete the [Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) and build practical ML engineering skills. It integrates theory, coding, software engineering, projects, data structures, and career preparation.

## Projects Summary

- **Music Mood Predictor** ‚Äî Linear regression on music features.
- **Meme Virality Predictor** ‚Äî Regression pipeline with preprocessing.
- **Anime Character Classifier** ‚Äî Binary classifier with logistic regression.
- **Pet Breed Classifier** ‚Äî Tiny CNN for image classification.
- **Climate Change Tweet Classifier** ‚Äî Ensemble learning with Random Forest/XGBoost.
- **Tourist Destination Recommender** ‚Äî Recommendation system using user profiles.
- **Recipe Similarity Engine** ‚Äî K-Means clustering + PCA for visualization.
- **Art Style Classifier** ‚Äî Image-based collaborative filtering system.
- **Mental Health Journal Analyzer** ‚Äî Sentiment prediction via Q-learning.
- **Smart City Traffic Predictor** ‚Äî CNN with TensorFlow and congestion datasets.
- **YouTube Video Performance Predictor** ‚Äî Flask API for inference.
- **AI Resume Tuner** ‚Äî Resume enhancer with Docker + OpenAI.
- **Resume Tweaker + Portfolio** ‚Äî Final capstone deployed via Streamlit.
